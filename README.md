##TASK 1: Data Pipeline (ETL)
📌 Description:
This task involves building a data pipeline to automate the process of data preprocessing, transformation, and loading (ETL). The pipeline is developed using Pandas and Scikit-learn to handle real-world data efficiently.
Initially, raw data is loaded into the system. The data is then cleaned by handling missing values and correcting inconsistencies. After cleaning, categorical features are converted into numerical form using encoding techniques. Numerical features are further standardized using scaling methods to improve model performance.
Finally, the processed data is stored in a new file, making it ready for machine learning tasks. This pipeline ensures that data preparation is consistent, repeatable, and efficient.

## TASK 2: Deep Learning Model
📌 Description:
In this task, a deep learning model is developed for image classification using TensorFlow. The model is trained on the MNIST dataset, which consists of handwritten digit images.
The data is first normalized and reshaped to make it suitable for a Convolutional Neural Network (CNN). The CNN model is designed with multiple layers such as convolution, pooling, and dense layers to automatically extract features from images.
The model is then trained and evaluated for accuracy. Visualization techniques are used to plot training and validation accuracy and loss graphs. This helps in understanding the model’s performance and learning behavior.
This task demonstrates the application of deep learning in solving real-world image recognition problems.


## TASK 3: End-to-End Data Science Project
📌 Description:
This task focuses on building a complete data science project, from data processing to deployment. A machine learning model is created to predict house prices based on input features such as area and number of bedrooms.
The project starts with dataset creation and preprocessing. A Linear Regression model is trained using Scikit-learn. After training, the model is saved using the pickle library.
To make the model accessible, a Flask API is developed. The API allows users to send input data and receive predictions in real time. This demonstrates how machine learning models can be deployed as web services.
This task highlights the full lifecycle of a data science project, including model development and deployment.


## TASK 4: Optimization using PuLP
📌 Description:
In this task, a business problem is solved using Linear Programming with the help of the PuLP library.
The objective is to maximize profit by deciding the optimal number of products to manufacture under given constraints such as limited resources (e.g., machine time and labor).
Decision variables are defined to represent the quantity of each product. An objective function is created to maximize total profit. Constraints are added to ensure resource limits are not exceeded.
The PuLP solver is used to compute the optimal solution. The results provide the best production plan and maximum achievable profit.
This task demonstrates how optimization techniques can be applied to real-world decision-making problems.
